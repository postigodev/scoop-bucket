# postigodev Scoop Bucket

This is the proposed README for a future `postigodev/scoop-bucket` repository.

## Install

```powershell
scoop bucket add postigodev https://github.com/postigodev/scoop-bucket
scoop install koba
```

## Update

```powershell
scoop update koba
```

## Apps

- `koba`: Local-first Git workflow configurator for real repositories.
